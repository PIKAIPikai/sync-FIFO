# sync_fifo
Verilog 硬件描述语言实现的**参数化单时钟同步FIFO**模块，采用**模块化设计**（FIFO控制逻辑+独立同步RAM），是FPGA/ASIC数字设计中通用的单时钟域数据缓存IP核。

## 核心特性
1. **单时钟同步架构**：读写共用同一时钟，适合同时钟域数据缓冲
2. **全参数化配置**：FIFO深度(`DEPTH`)、数据位宽(`WIDTH`)可灵活修改
3. **内置同步RAM**：独立封装`sync_ram`存储模块，解耦设计，易复用维护
4. **智能位宽计算**：内置`clogb2`函数，自动计算地址位宽，无需手动修改
5. **经典判空满逻辑**：二进制扩展指针实现空/满标志，资源占用低、时序稳定
6. **安全读写保护**：读写使能联动空满标志，避免数据溢出/读空异常
7. **异步复位初始化**：低电平异步复位，上电自动清零存储单元
8. **通用可综合**：纯RTL实现，无厂商IP依赖，兼容Xilinx/Intel全系列FPGA

## 参数配置
| 参数名   | 功能           | 默认值 |
| -------- | -------------- | ------ |
| DEPTH    | FIFO存储深度   | 64     |
| WIDTH    | 数据位宽       | 8      |

## 端口信号定义
| 端口名   | 方向   | 功能                     |
| -------- | ------ | ------------------------ |
| clk      | 输入   | 同步工作时钟（单时钟）|
| rst_n    | 输入   | 异步低电平复位           |
| wren     | 输入   | 写使能信号（高电平有效） |
| wdata    | 输入   | 写数据总线               |
| rden     | 输入   | 读使能信号（高电平有效） |
| rdata    | 输出   | 读数据总线               |
| full     | 输出   | FIFO满标志（高电平有效） |
| empty    | 输出   | FIFO空标志（高电平有效） |

## 代码架构
本项目采用**两层模块化设计**，逻辑清晰、复用性强：
1. **sync_fifo**：顶层模块，实现读写指针控制、空满标志判断、RAM例化
2. **sync_ram**：底层存储模块，单时钟同步RAM，负责数据读写存储

## 设计优势
